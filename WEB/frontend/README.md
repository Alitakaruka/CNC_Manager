# 🖨️ CNCManager Pro

Современное веб-приложение для управления 3D-принтерами с красивым интерфейсом и расширенной функциональностью.

## ✨ Особенности

- 🎨 **Современный дизайн** - Красивый UI с анимациями и градиентами
- 📱 **Адаптивный интерфейс** - Работает на всех устройствах
- 🔌 **Множественные подключения** - Поддержка COM, IP и USB подключений
- 📊 **Реальное время** - Мониторинг статуса принтеров в реальном времени
- 🎮 **Управление** - Полный контроль над движением и командами принтера
- 📁 **Загрузка файлов** - Поддержка G-code файлов для печати
- 📝 **Логирование** - Детальные логи с экспортом
- 🔔 **Уведомления** - Toast-уведомления для всех действий

## 🚀 Технологии

- **React 19** - Современная библиотека для UI
- **Vite** - Быстрый сборщик проекта
- **Tailwind CSS** - Утилитарный CSS фреймворк
- **Framer Motion** - Анимации и переходы
- **Lucide React** - Красивые иконки
- **React Hot Toast** - Уведомления

## 📦 Установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd frontend

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## 🎯 Функциональность

### Подключение принтеров
- **COM порты** - Подключение через последовательные порты
- **IP адреса** - Сетевое подключение с указанием IP и порта
- **USB** - Прямое USB подключение

### Управление принтерами
- **Движение по осям** - X, Y, Z, E с настраиваемым шагом
- **Быстрые команды** - Home, Disable Motors и другие
- **Пользовательский G-code** - Отправка любых G-code команд
- **Температурный контроль** - Мониторинг сопла и стола

### Мониторинг
- **Статус принтеров** - Онлайн/офлайн/печать
- **Прогресс печати** - Процент выполнения и время
- **Температуры** - Реальное время сопла и стола
- **Логи** - Детальная история операций

## 🔧 API Endpoints

- `GET /api/Printers` - Получение списка принтеров
- `POST /api/StartPrint` - Запуск печати
- `POST /api/sendGCode` - Отправка G-code команд

## 📁 Структура проекта

```
src/
├── components/
│   ├── Buttons/          # Кнопки и элементы управления
│   ├── Connection/       # Компоненты подключения
│   ├── Header/          # Шапка приложения
│   ├── ModalWindows/    # Модальные окна
│   ├── Pages/           # Страницы приложения
│   ├── PrinterData/     # Компоненты данных принтера
│   └── UserInfo/        # Уведомления и информация

└── assets/             # Статические ресурсы
```

Все анимации реализованы с помощью Framer Motion для плавных переходов.
